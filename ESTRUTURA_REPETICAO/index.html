Estrutura de Repetição

O que são estruturas de Repetição?

  • A ideia é repetir uma ação até atingir uma condição
  • Ao invés de repetirmos o mesmo código várias vezes, criamos um statement que fará uma checagem em cada loop
  • Também chamada de loop

Estruta de Repetição: while

  • Fará uma ação, até que a condição seja atingida
  • Precisamos realmente 'definir um fim' para o loop, caso contrário, ocorrerá o problema de 'loop infinito'

Exemplo:

let x = 10

while(x > 0) {
console.log(x)
x = x -1
}

let y = 0
while (y <= 10) {
console.log(y) y = y + 1
}
let z = 0
while (z <= 10) {
console.log(z)
z = z + 3
}

Estruta de Repetição: do while

  • Semelhante ao while, porém a estrutura muda um pouco
  • Este método quase não é utilizado
let a = 100

do{ console.log(a/2)
a = a - 5
}while(a >=0)

Estrutura de repetição: for

  • Semelhante ao wile, porém amplamente utilizado
  • Estrutura de repetição mais fácil de compreender
for (let numero = 2; numero < 100; numero = numero * 2) {
console.log('O número é ' + numero)
}
for (let i = 0; i < 100; i = i + 15) {
console.log(`A soma de i com 2 é: ${1 + 2}`)
}

Forçando a saída de um loop: break

  • As vezes precisamos parar o loop antes que complete todo seu ciclo
  • Para isso, utilizamos o break
for(let i = 5; i < 20; i = i + 1){
if(i % 10 == 0) {
console.log('Saiu do loop')
break;
}
console.log('Prosseguindo o loop')
}

let nome = 'Mauricio'
for(let i = 0; i < 10; i = i + 1){
if(i == 3){
nome='Mauricio Rodrigues'
}
if(i == 8 && nome == 'Mauricio Rodrigues') {
console.log('O nome é ' + nome + ', pode parar')
break
}
console.log(i)
}


Pulando uma execução do loop: continue

  • Dependendo da nossa lógica, podemos pular o resto da execução
  • Para isso, utilizamos a palavra continue
let x = 0
while(x < 11) {
x = x + 1
if( x % 2 == 0) {
continue;
}
console.log(x)
x = x + 1
}

for(let i = 10; i > 0; i = i - 1){
if(i % 2 == 0){
console.log('Caiu no continue')
continue
}
console.log(i)
}
