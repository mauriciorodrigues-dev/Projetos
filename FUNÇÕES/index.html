O que são Funções?

  • Funções são estruturas de códigos menores que são reaproveitados durante a execução/construção de um programa
  • Principal objetivo: poupar repetição de código
  • Podem ser consideradas 'subprogramas'

Definindo uma função

  • Uma função tem uma estrutura um pouco mais complexa
  • Devemos declarar a função sempre com a palavra 'function'
  • Uma função deve ter um nome
  • Pode conter argumentos/parametros, definidos entre ()
  • O corpor da função é definido entre {}
  • Geralmente uma função retorna um valor
  • É psssível declarar funções em variáveis

Exemplos

Função sem argumentos

function imprimirNoConsole() {
console.log('Olá Mundo')
}

imprimirNoConsole()

Função com argumentos

function imprimirUmNumero(num){
console.log('O número é: ' + num)
}
imprimirUmNumero(2)
imprimirUmNumero(4)
imprimirUmNumero(5)

Função Anônima atrelada a uma variável

const numeroAleatorio = function(){
console.log(Math.random())
}

numeroAleatorio()
numeroAleatorio()
